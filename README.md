## Description

Keep Moving Forward is a game that aims to build empathy for people living with depression. Players control Rachel as she navigates a maze to collect three keys and progress to the next level. However, Rachel’s energy randomly depletes, slowing her movement and darkening her vision to represent the fatigue commonly experienced by those with depression. Throughout the maze, players can find hearts that restore Rachel’s energy, symbolizing the importance of support and care from others.

## Setup and Interaction Instructions

- Open Github link to launch the game
- Use arrow keys or WASD keys to move
- Press 'R' to reset
